# Amazon-E-Commerce-Intelligence-Dashboard

An interactive Power BI dashboard designed to analyze Amazon e-commerce sales, customer behavior, product category performance, and delivery efficiency. The project transforms raw transaction data into actionable business insights through KPI monitoring and dynamic visualizations.

## Tools & Technologies

* Power BI
* Power Query
* DAX
* Data Modeling
* Bookmarks
* Hidden Slicers

## Key Features

* Revenue & Sales Analysis
* Category Performance Analysis
* Customer Satisfaction Tracking
* Delivery Performance Monitoring
* Discount Efficiency Analysis
* Interactive Navigation using Bookmarks
* Dynamic Filtering using Hidden Slicers

## KPIs

* Total Revenue
* Total Orders
* Average Order Value (AOV)
* Return Rate
* Delivery Success Rate
* Customer Satisfaction Score

## Key Insights

* Electronics generated the highest revenue and customer satisfaction.
* Revenue showed strong growth from 2024 to 2025.
* Home and Sports categories contributed significantly to overall sales.
* Beauty and Clothing categories present growth opportunities.
* Delivery performance directly impacts customer experience.
